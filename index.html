talking-text takes text from your webpage and types it out just like those classic games you love so much!

It's great for webcomics, interactive demos or anything you'd like to give that unmistakeable ‘gamelike’ feeling.

But the fun doesn't stop there: please allow this helpful frog to show you just some of the options talking-text offers (you'll have to nudge him first — he can be as lazy as he can be helpful):

      'p.loud-slow-wizard').talkingText({
      // Change speed of typing
      pace: 100,
      // Customise 'voice' provided by Web Audio API

      1, pitch:
      100, volume:
      1 }

      callback:

      'Finished typing: ' + text); } }); })
